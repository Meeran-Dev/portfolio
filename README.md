# Meeran.Dev - Portfolio Desktop

A pixel-art desktop OS-style portfolio

## Features
- Movable & resizable windows (drag titlebar, resize from any edge/corner)
- Boot screen with loading sequence
- Taskbar with live clock + open window items
- Right-click context menu on desktop
- Interactive terminal with command history (↑↓ arrows)
- Minimize · Maximize · Close window controls

## Terminal commands

| Command | Output |
|---|---|
| `help` | list all commands |
| `about` | short bio |
| `ls` | list desktop files |
| `ping github` | get GitHub URL |
| `uname -a` | system info |
| `sudo hire me` | 😏 |
| `clear` (or Ctrl+L) | clear terminal |
